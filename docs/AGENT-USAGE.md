# Using Perchance AI Toolkit with AI Agents

This guide explains how to connect Claude Code, OpenClaw, or any MCP-compatible agent to the Perchance AI Toolkit.

---

## What the agent can do

Once connected, your agent can:
- **Generate** complete `.perchance` generators from any topic using AI
- **Browse** 150+ built-in templates by category
- **Validate** `.perchance` syntax before running
- **Preview** results locally (instant, no browser)
- **Run live** on perchance.ai via Playwright (automated browser)
- **Scrape** and clone any public perchance.ai generator

The agent handles the full pipeline: generate → validate → preview → run → save. You never need to open a browser.

---

## Claude Desktop Setup

### 1. Install the MCP server
```bash
npm install -g perchance-ai-prompt-library
```

### 2. Add to Claude Desktop config
Location: `~/Library/Application Support/Claude/claude_desktop_config.json` (macOS)
or `%APPDATA%\Claude\claude_desktop_config.json` (Windows)

```json
{
  "mcpServers": {
    "perchance": {
      "command": "npx",
      "args": ["perchance-mcp"],
      "env": {
        "GROQ_API_KEY": "your-groq-api-key-here"
      }
    }
  }
}
```

### 3. Restart Claude Desktop

You should see a 🔌 icon indicating the MCP server is connected.

### 4. Talk to Claude

Example prompts:
- *"Create a fantasy character name generator with rare/common/legendary tiers"*
- *"Generate 30 sci-fi weapon descriptions and save them"*
- *"Scrape the generator at perchance.ai/fantasy-name-gen and adapt it for a cyberpunk setting"*
- *"Show me all the dialogue templates you have"*

## Available MCP Tools

| Tool | Description | Required params |
|------|-------------|----------------|
| `generate_perchance` | AI-generate a `.perchance` generator | `topic` |
| `list_templates` | List 150+ templates by category | — |
| `get_template` | Get full code of a template | `name` |
| `validate_syntax` | Validate `.perchance` code | `code` |
| `preview_rolls` | Preview results locally | `code` |
| `run_on_perchance` | Run on perchance.ai via Playwright | `code` |

## Agent Workflows

### Create Generator (full pipeline)
```
Agent receives: "fantasy tavern name"
  → generate_perchance(topic: "fantasy tavern name", style: "weighted")
  → validate_syntax(code)
  → preview_rolls(code, count: 10)
  → run_on_perchance(code, rolls: 20, screenshot: true)
  → saves to output/generators/fantasy-tavern-name.perchance
  → returns results + screenshot to you
```

### Improve Existing Generator
```
Agent receives: [your .perchance code] + "add more variety"
  → validate_syntax(existing code)
  → AI rewrites with improvements
  → preview_rolls(new code)
  → returns improved code
```

### Scrape & Clone
```
Agent receives: perchance.ai URL
  → run_on_perchance(scrapeUrl: url)
  → extracts source code
  → adapts per your instructions
  → validates + previews
  → saves to output/cloned/
```

## Troubleshooting

**MCP server not connecting?**
- Make sure `npm install -g perchance-ai-prompt-library` ran successfully
- Verify `GROQ_API_KEY` is set in the config env
- Restart Claude Desktop after config changes

**Playwright not working?**
```bash
npm install playwright
npx playwright install chromium
```

**Generator has no `output` list warning?**
Perchance.ai needs a list named `output` to display results. The validator will warn you. Fix: rename your first list to `output`.
