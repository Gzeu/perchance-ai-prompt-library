# ⚡ Perchance AI Prompt Library

**The most advanced AI-powered Perchance.org generator toolkit** with an **Ultra Agentic Brainstorm System** featuring specialized autonomous agents, dynamic skills, collaborative voting, persistent memory, and a rich categorized template library (150+ templates).

Generate production-quality Perchance generators from natural language using Groq + multi-agent collaboration.

## ✨ Core Features

- **Ultra Agentic Brainstorm Engine** (v7.0): Up to 7 specialized agents collaborate, debate, vote with weighted stars, and iterate up to 4 rounds.
- **Dynamic Skill System**: Agents scored on Narrative Depth, Syntactic Perfection, Originality, Consistency, Creativity, etc.
- **Intelligent Agent Assembly**: Auto-selects best agents based on request category.
- **Persistent Memory**: Long-term session memory + knowledge base to avoid repetition.
- **Rich Template Library**: 150+ high-quality templates across 12 categories.
- **Full Stack Support**: CLI, Web Studio (Ultra Agentic tab), REST API, Discord Bot.

## 🚀 Quick Start

```bash
git clone https://github.com/Gzeu/perchance-ai-prompt-library.git
cd perchance-ai-prompt-library
npm install
npm run dev
```

Open `http://localhost:5173` and go to **Ultra Agentic Studio** tab.

## 🧠 Ultra Agentic Mode

1. Describe your idea in natural language.
2. System assembles relevant agents.
3. Agents generate variants → Debate → Weighted Voting.
4. Iterative refinement.
5. Export best Perchance generator.
