Powerful Generation

Available Endpoints

GET /api/health

Check if the API is running and healthy.

GET /api/styles

Get a list of all available art styles.

POST /api/prompts/generate

Generate a new AI art prompt based on the selected style and parameters.

GET /api-docs

Interactive API documentation with Swagger UI.
